### My Projects

Here's a glimpse of some of my key projects:

#### I. Core Web Applications:

##### **Brightnify: Household Services Application**
* **Description:** A multi-user platform (Admin, Service Professional, Customer) designed for comprehensive home servicing.
* **Tech:** Python, Flask, Jinja2, Bootstrap, SQLite.
* **Key Learnings:** Gained foundational experience in multi-role access control, CRUD operations, and core Flask web development principles.

##### **QuizXpert: Exam Preparation Platform**
* **Description:** A robust multi-user platform (Admin/Quiz Master, User) for creating and attempting quizzes, featuring automated engagement mechanisms.
* **Tech:** Python, Flask API, Vue.js, SQLite, Redis, Celery, Flask-Security/JWT, Bootstrap.
* **Key Learnings:** Developed expertise in decoupled architecture (frontend/backend), designing RESTful APIs, implementing asynchronous tasks (Celery/Redis), utilizing caching strategies, scheduling jobs, and building robust authentication/authorization systems.

#### II. AI/Generative AI Applications:

##### **EduWise – AI-based Learning Aid**
* **Description:** An AI-powered tool designed to assist learning by generating educational content, quizzes, audio explanations, and flashcards.
* **Tech:** Python, Gemini API, PDF/PPT generation, Speech Synthesis.
* **Key Learnings:** Integrated the Gemini API for content generation and explored multi-modal output capabilities (text, audio, document generation).

##### **GenAI Resume Optimizer**
* **Description:** A powerful tool that takes a job description and a resume to provide an ATS score, salary range prediction, job match analysis, tailoring recommendations, skill suggestions, and resume order advice.
* **Tech:** Python, Gemini API, Streamlit, PDF/DOCX parsing.
* **Key Learnings:** Leveraged the Gemini API for complex Natural Language Processing (NLP) and analysis, built interactive AI demos using Streamlit, and implemented document parsing for practical GenAI applications in career development.

##### **News Personalized & Automated Mailer**
* **Description:** A system that scrapes news based on user preferences, summarizes it using Gemini, and sends personalized daily emails with original links.
* **Tech:** Python, Gemini API, Web Scraping (Google News), Email Sending, Flask.
* **Key Learnings:** Gained experience in web scraping, utilizing the Gemini API for summarization, automating email delivery, and creating personalized content streams.

#### III. Basic/Foundation Projects:

##### **Face-Based Attendance System**
* **Description:** A system that marks attendance using face recognition, storing data in XML files.
* **Tech:** Python, OpenCV.
* **Key Learnings:** Applied Computer Vision (OpenCV) fundamentals, practiced basic file I/O operations (XML), and implemented interactive elements like flash messages and voice outputs.
